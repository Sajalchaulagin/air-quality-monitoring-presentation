Introduction

Our project is designed to monitor air quality from multiple locations using low-cost IoT hardware. We deployed two ESP32-based sensor nodes equipped with DHT11, MQ135, MQ2, and MQ3 sensors. Data is uploaded to Firebase Realtime Database and visualized on a live cloud dashboard.
